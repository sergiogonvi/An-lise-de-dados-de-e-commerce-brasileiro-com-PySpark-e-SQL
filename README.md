# 📊 Análise de E-commerce Brasileiro — Olist

Projeto de análise de dados em **Python + PySpark + SQL**, usando dados reais
de um marketplace brasileiro com mais de 100 mil pedidos.

## 🎯 Objetivo
Responder perguntas de negócio reais sobre vendas, logística de entrega e
satisfação do cliente, usando processamento distribuído com PySpark.

## 🛠️ Tecnologias
- Python 3
- PySpark (DataFrame API + Spark SQL)
- SQL (joins, agregações, window functions)
- Matplotlib / Seaborn
- Google Colab

## 📦 Dataset
[Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce) — Kaggle

## 📌 Perguntas de negócio respondidas
1. Quais estados brasileiros mais compram (em faturamento)?
2. Quais categorias de produto têm a melhor avaliação média?
3. O tempo de entrega afeta a satisfação do cliente?
4. Como evoluiu o faturamento mês a mês?
5. Quais categorias têm os piores atrasos de entrega?

## 📈 Principais insights
- Forte concentração de faturamento em poucos estados
- Relação clara entre tempo de entrega e nota de avaliação do cliente
- Identificação de categorias com maior risco de atraso logístico
- Sazonalidade visível ao longo dos meses analisados

## 🚀 Como rodar
1. Abra o notebook `analise_ecommerce_olist.ipynb` no [Google Colab](https://colab.research.google.com)
2. Crie uma conta gratuita no [Kaggle](https://www.kaggle.com) e gere seu próprio token de API
3. Cole seu token na célula indicada e execute as células em ordem
